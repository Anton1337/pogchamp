# pogchamp

A blazing fast utils library.



## Installation

Install pogchamp with npm

```bash
  npm install pogchamp
```

Install pogchamp with yarn

```bash
  yarn add pogchamp
```

Install pogchamp with pnpm

```bash
  pnpm add pogchamp
```
